# ArogyaAI

ArogyaAI is a digital health assistant that helps users digitize and understand medical reports.  
Users can upload medical documents, extract text using OCR, and interact with an AI chatbot for support.

## Features
- Medical document OCR extraction
- AI chatbot for health related queries
- Multilingual support
- Health profile management

## Tech Stack
Frontend
- Next.js
- React
- TypeScript
Backend
- Python
- FastAPI
AI
- Llama3 model (via Groq API)
- Tesseract.js
## Running the Project
1. Clone the repository
git clone https://github.com/Aishwarya3306/DesignProject2.git
2. Go to the project folder
cd arogya-ai
3. Install dependencies
npm install
4. Run the project

npx next dev
